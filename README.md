# AI Agent Decision Debugger

**AI Agent Decision Debugger** es una herramienta de observabilidad en tiempo real diseñada para rastrear la lógica de decisión y la ejecución de herramientas de agentes de IA.

## Función Principal

La función principal de este proyecto es proporcionar una interfaz visual que permita a los desarrolladores:
- **Rastrear Decisiones**: Visualizar el proceso de pensamiento y razonamiento detrás de cada acción tomada por un Agente de IA.
- **Monitorear Ejecución de Herramientas**: Ver qué herramientas se están utilizando, cuándo y con qué parámetros.
- **Observabilidad Total**: Obtener una visión clara y detallada del flujo de ejecución en tiempo real para facilitar la depuración y optimización.

## Tecnologías

- **Next.js**: Framework principal para la interfaz de usuario.
- **TypeScript**: Para un desarrollo robusto y tipado.
- **Vanilla CSS**: Estilizado personalizado y flexible.

## Inicio Rápido

```bash
npm install
npm run dev
```
