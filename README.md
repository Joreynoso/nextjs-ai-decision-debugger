# 🛠️ AI Agent Decision Debugger

**AI Agent Decision Debugger** is a premium, real-time observability platform designed to track, visualize, and debug the decision-making logic of AI agents. It provides a cinematic interface for developers to monitor tool execution, reasoning paths, and agent behaviors.

---

## 🚀 Key Features

- **Real-time Trace Visualizer**: Live monitoring of agent thought processes and tool calls.
- **Decision Logic Tracking**: Deep dive into the reasoning behind every agent action.
- **Premium Glassmorphism UI**: A state-of-the-art interface built for modern developers.
- **Tool Execution Monitoring**: Track parameters, outputs, and execution timing.
- **Workflow Visualization**: Interactive flowcharts of agent logic using modern animations.

## 🧰 Extended Tech Stack

This project leverages cutting-edge web technologies to deliver a high-performance, visually stunning experience:

### Core Frameworks
- **Next.js 15.2.1 (App Router)**: Hybrid static/server rendering power.
- **React 19**: Utilizing the latest concurrent features and hooks.
- **TypeScript**: Full type safety for reliable, enterprise-grade development.

### UI & Styling
- **Tailwind CSS 4**: Modern styling with the latest JIT engine refinements.
- **Framer Motion**: Smooth, cinematic micro-animations and page transitions.
- **Radix UI**: Primitive components for accessible, high-quality user interfaces.
- **Lucide React**: Beautifully crafted icons for a premium visual feel.
- **Shadcn UI**: Pre-designed, highly customizable UI components.

### Quality & Architecture
- **ESLint 9**: Modern linting for consistent code quality.
- **PostCSS 8**: Advanced CSS transformations and optimizations.

## 🛠️ Getting Started

1. **Clone and Install**:
```bash
git clone https://github.com/Joreynoso/nextjs-ai-decision-debugger.git
cd nextjs-ai-decision-debugger
npm install
```

2. **Development Mode**:
```bash
npm run dev
```

3. **Build Target**:
```bash
npm run build
```
